# ⚛️ PhysicsFormulsBot

**Бот-репетитор по физике**, который помогает студентам и школьникам готовиться к экзаменам.  
Больше не нужно перерывать сотни сайтов — всё, что нужно учить, уже собрано в одном месте.

> 🚀 **В будущем:**  
> Появятся практические материалы для закрепления знаний, а также возможность использовать бота на своём хостинге для помощи учащимся.

---

## 📚 Что умеет бот

Бот выдаёт **формулы по физике** и позволяет администраторам пополнять базу знаний.

### 🤖 Доступные команды

| Команда | Описание |
|---------|----------|
| `/tutorial_names` | Все физические обозначения (символы и их значение) |
| `/physics_formuls` | Поиск формул для указанного физического обозначения |
| `/help` | Список всех команд, которые знает бот |
| `/support` | Контакты администраторов для связи и решения проблем |
| `/append_formul` | ➕ Добавление новых формул *(только для администраторов)* |

---

## 👑 Администрирование

Администраторы бота задаются в файле `config.py` — **атрибут `ID_ADMIN`**.

> ℹ️ Именно администраторы могут пополнять базу формул через команду `/append_formul`.

---

## 🗄️ База данных

К боту прилагается готовая БД `formuls.sqlite`, содержащая **все необходимые формулы для подготовки к экзаменам по физике**, включая:

- школьную программу  
- подготовку к **ЕГЭ**  
- и другие вступительные испытания

**PhysicsFormulsBot** — это удобный инструмент для систематизации знаний по физике.
